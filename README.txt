There are 4 screens that you can naviage from and to. 

1. Home Screen: Directions on how to use web app, as well as provides your coordinates.
2. Grade Distributions Screen: This screen reuqires the Grade Distribution API that I created to be on, so if the grade data is not loading, the API is off. 
3. Weather Screen: This page shows the weather for specific chicago coordinates. This is also calling an API to get the weather data info.
4. Stored Data: This screen will display all the search history for your searches on the Grade Distributions page.
